此项目是保利威鸿蒙多场景 Demo。

此项目支持视频播放、在线聊天、打赏、商品、互动等功能（具体见[多场景支持的功能特性](https://github.com/polyv/polyv-harmony-livescenes-sdk-demo/blob/master/docs/public/支持的功能特性.md)）

多场景项目的文件目录结构如下：

```
|-- entry
|   |-- ability
|   |   `-- PLVEntryAbility (SDK初始化)
|   |-- pages
|   |   `-- PLVLoginPage (登录页)
|   `-- startup (播放器初始化)
|-- scenes_live (直播带货场景模块)
|   |-- pages (观看页)
|   `-- components (功能组件)
`-- scenes_liveWatch (观看端场景模块)
    |-- common (场景内通用组件)
    |-- components (功能组件)
    `-- pages (观看页)
```
